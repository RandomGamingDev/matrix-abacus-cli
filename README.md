# Matrix Abacus CLI
### Just CLI for accurately dealing with Matrix Operations without being a matrix calculator which solves everything or clunky
Great for double checking work and is basically a CLI port of https://textbooks.math.gatech.edu/ila/demos/rrinter.html for quick & efficient use.

It's **not** a matrix calculator and purposely not so since those are clunky, slow, and oftentimes do more than you should have access to. <br/>
This is an **abacus** in the sense that you're only given **basic operations** (i.e. swapping, multiplying, replacement) and is primarily for **verifying work** when or after doing stuff on paper.
